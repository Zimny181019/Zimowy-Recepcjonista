🔐 Privacy Policy

🤖 Bot: Zimowy Recepcjonista
📅 Effective Date: July 3, 2026
This Privacy Policy explains how the bot processes Discord data.

📋 Data Collection

  • 👤 User IDs
  • 🏠 Server (Guild) IDs
  • 💬 Channel IDs
  • 🎭 Role IDs
  • 🟢 Presence status (online / offline)

The bot does not read, log, or store message content. No Message Content Intent is used or requested. Presence status is used only in aggregate to count how many admins are currently online and is not stored long-term or shared with third parties.

⚙️ Usage

  • 📊 Member & admin statistics (live counter channels)
  • 🟢 Admin online tracking
  • ✅ User verification (role assignment)
  • 👋 Welcome & goodbye messages
  • 🤖 Automation (slash commands, event handling)

Collected identifiers (User IDs, Role IDs, Channel IDs, Server IDs) are used solely to operate these features on the server where the bot is installed. Data is stored locally in the bot's database only for as long as needed to provide these features, and is removed when a user leaves the server.

📩 Contact

If you have questions about this Privacy Policy or wish to request removal of your data, please contact the server administrators.
